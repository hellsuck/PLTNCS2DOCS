# Memory API

The `memory` table provides functions for low-level memory access and validation.

---

### `memory.read_i32`
Reads a 32-bit signed integer from the given address.

<Property name="addr" type="uintptr_t" required>
  The memory address to read from.
</Property>

**Returns**: `int`

```lua
local health = memory.read_i32(pawn_address + 0x32C)
```

---

### `memory.read_u32`
Reads a 32-bit unsigned integer from the given address.

<Property name="addr" type="uintptr_t" required>
  The memory address to read from.
</Property>

**Returns**: `uint`

```lua
local flags = memory.read_u32(pawn_address + 0x3C8)
```

---

### `memory.read_f32`
Reads a 32-bit float from the given address.

<Property name="addr" type="uintptr_t" required>
  The memory address to read from.
</Property>

**Returns**: `float`

```lua
local flash_duration = memory.read_f32(pawn_address + 0x1468)
```

---

### `memory.read_ptr`
Reads a pointer (uint64) from the given address.

<Property name="addr" type="uintptr_t" required>
  The memory address to read from.
</Property>

**Returns**: `uintptr_t`

```lua
local node = memory.read_ptr(pawn_address + 0x10)
```

---

### `memory.is_valid`
Checks if a memory address is valid and accessible.

<Property name="addr" type="uintptr_t" required>
  The memory address to validate.
</Property>

**Returns**: `bool`

```lua
if memory.is_valid(pawn_address) then
    local health = memory.read_i32(pawn_address + 0x32C)
end
```

---

### `memory.get_module_base`
Returns the base address of a loaded module.

<Property name="name" type="string" required>
  The name of the module (e.g., `'client.dll'`).
</Property>

**Returns**: `uintptr_t` (the base address of the module)

```lua
local client_base = memory.get_module_base("client.dll")
if client_base ~= 0 then
    print("client.dll is loaded at: " .. tostring(client_base))
end
```
