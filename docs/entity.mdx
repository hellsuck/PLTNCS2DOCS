# Entity API

The `entity` table provides functions for interacting with game entities in Counter-Strike 2.

---

### `entity.get_local_player`
Returns the local player controller pointer.

**Returns**: `uintptr_t`

```lua
local local_player = entity.get_local_player()
if local_player ~= nil then
    print("Local player pointer: " .. tostring(local_player))
end
```

---

### `entity.get_local_player_pawn()`
Returns the local player pawn pointer (`uintptr_t`).

---

### `entity.get_player_controller`
Returns a player controller pointer by index.

<Property name="index" type="int" required>
  The player index (starting from 1).
</Property>

**Returns**: `uintptr_t`

```lua
local player_one = entity.get_player_controller(1)
```

---

### `entity.get_pawn(controller)`
Returns the pawn associated with a player controller.
- **Parameters**: 
    - `controller` (`uintptr_t`): The controller pointer.

---

### `entity.get_team`
Returns the entity's team ID.

<Property name="ent" type="uintptr_t" required>
  The entity pointer (player pawn or controller).
</Property>

**Returns**: `int` (2 for T, 3 for CT, 1 for Spectator)

```lua
local team = entity.get_team(entity.get_local_player_pawn())
if team == 2 then
    print("You are on Terrorists")
elseif team == 3 then
    print("You are on Counter-Terrorists")
end
```

---

### `entity.get_origin(ent)`
Returns the entity's world position as a `Vector3` object.
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_view_offset(ent)`
Returns the entity's view offset (relative to origin) as a `Vector3` object.
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_eye_position(ent)`
Returns the entity's absolute eye position as a `Vector3` object.
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_health`
Returns the entity's current health.

<Property name="ent" type="uintptr_t" required>
  The entity pointer.
</Property>

**Returns**: `int`

```lua
local lp_pawn = entity.get_local_player_pawn()
local health = entity.get_health(lp_pawn)
print("My health: " .. health)
```

---

### `entity.is_alive(ent)`
Returns `true` if the entity is alive.
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_class_name(ent)`
Returns the internal class name of the entity.
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_network_name(ent)`
Returns the network class name (e.g., `'C_CSPlayerPawn'`).
- **Parameters**: 
    - `ent` (`uintptr_t`): The entity pointer.

---

### `entity.get_all()`
Returns a table containing all active entity pointers.

---

### `entity.get_by_class`
Returns a table of entities matching a Network Name.

<Property name="className" type="string" required>
  The network class name (e.g., `'C_CSPlayerPawn'`).
</Property>

**Returns**: `table` of `uintptr_t`

---

### `entity.get_players()`
Returns a table of all valid player controller entity pointers.

```lua
local players = entity.get_players()
for i, player in ipairs(players) do
    local health = entity.get_health(player)
    print("Player index " .. i .. " has " .. health .. " HP")
end
```
