# Draw API

The `draw` table provides primitive rendering functions for creating overlays and UI elements.

---

### `draw.line`
Draws a line on the screen.

<Property name="x1" type="float" required>
  The X coordinate of the starting point.
</Property>

<Property name="y1" type="float" required>
  The Y coordinate of the starting point.
</Property>

<Property name="x2" type="float" required>
  The X coordinate of the ending point.
</Property>

<Property name="y2" type="float" required>
  The Y coordinate of the ending point.
</Property>

<Property name="col" type="color_t" required>
  The color object `{r, g, b, a}`.
</Property>

<Property name="thick" type="float" required>
  The thickness of the line.
</Property>

```lua
draw.line(100, 100, 200, 200, {255, 255, 255, 255}, 1.0)
```

---

### `draw.rect`
Draws a hollow rectangle.

<Property name="x" type="float" required>
  The X coordinate of the top-left corner.
</Property>

<Property name="y" type="float" required>
  The Y coordinate of the top-left corner.
</Property>

<Property name="w" type="float" required>
  The width of the rectangle.
</Property>

<Property name="h" type="float" required>
  The height of the rectangle.
</Property>

<Property name="col" type="color_t" required>
  The color object `{r, g, b, a}`.
</Property>

<Property name="round" type="float" required>
  The rounding radius of the corners.
</Property>

<Property name="thick" type="float" required>
  The thickness of the rectangle border.
</Property>

```lua
draw.rect(50, 50, 100, 100, {255, 0, 0, 255}, 5.0, 2.0)
```

---

### `draw.rect_filled(x, y, w, h, col, round)`
Draws a filled rectangle.
- **Parameters**: 
    - `x, y` (`float`): Top-left coordinates.
    - `w, h` (`float`): Width and height.
    - `col` (`color_t`): Color object.
    - `round` (`float`): Corner rounding radius.

---

### `draw.rect_filled_multi_color(x, y, w, h, col_tl, col_tr, col_br, col_bl)`
Draws a rectangle with per-corner colors.
- **Parameters**: 
    - `x, y, w, h`: Rectangle dimensions.
    - `col_tl` (`color_t`): Top-left color.
    - `col_tr` (`color_t`): Top-right color.
    - `col_br` (`color_t`): Bottom-right color.
    - `col_bl` (`color_t`): Bottom-left color.

---

### `draw.circle(x, y, radius, col, segments, thick)`
Draws a hollow circle.
- **Parameters**: 
    - `x, y` (`float`): Center coordinates.
    - `radius` (`float`): Circle radius.
    - `col` (`color_t`): Color object.
    - `segments` (`int`): Number of segments for the circle.
    - `thick` (`float`): Line thickness.

---

### `draw.circle_filled(x, y, radius, col, segments)`
Draws a filled circle.
- **Parameters**: 
    - `x, y` (`float`): Center coordinates.
    - `radius` (`float`): Circle radius.
    - `col` (`color_t`): Color object.
    - `segments` (`int`): Number of segments for the circle.

```lua
draw.circle_filled(400, 400, 20, {0, 255, 0, 255}, 32)
```

---

### `draw.triangle(x1, y1, x2, y2, x3, y3, col, thick)`
Draws a hollow triangle.
- **Parameters**: 
    - `x1, y1, x2, y2, x3, y3`: Coordinates for the three vertices.
    - `col` (`color_t`): Color object.
    - `thick` (`float`): Line thickness.

---

### `draw.triangle_filled(x1, y1, x2, y2, x3, y3, col)`
Draws a filled triangle.
- **Parameters**: 
    - `x1, y1, x2, y2, x3, y3`: Coordinates for the three vertices.
    - `col` (`color_t`): Color object.

---

### `draw.box(x, y, w, h, col, thick)`
Draws a standard 2D box.
- **Parameters**: 
    - `x, y` (`float`): Top-left coordinates.
    - `w, h` (`float`): Width and height.
    - `col` (`color_t`): Color object.
    - `thick` (`float`): Line thickness.

```lua
draw.box(300, 300, 50, 100, {0, 0, 255, 255}, 1.5)
```

---

### `draw.box_corner(x, y, w, h, col, thick, length)`
Draws a 2D corner-style box. Useful for ESP visuals.
- **Parameters**: 
    - `x, y, w, h, col, thick`: Standard dimensions and styling.
    - `length` (`float`): Length of the corner lines.

---

### `draw.text`
Draws text on the screen.

<Property name="x" type="float" required>
  The X coordinate for the text placement.
</Property>

<Property name="y" type="float" required>
  The Y coordinate for the text placement.
</Property>

<Property name="col" type="color_t" required>
  The color object `{r, g, b, a}`.
</Property>

<Property name="txt" type="string" required>
  The string to display.
</Property>

<Property name="outline" type="bool" required>
  If true, a black outline is drawn around the text.
</Property>

<Property name="center" type="bool" required>
  If true, the text is centered horizontally.
</Property>

```lua
draw.text(960, 540, {255, 255, 255, 255}, "Hello Friend, Maybe I should give you a name", true, true)
```
