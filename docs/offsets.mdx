# Offsets

The `offsets` table contains pre-defined memory offsets for Counter-Strike 2. These offsets are updated automatically by the engine.

---

## `offsets.client_dll`

This table contains offsets relevant to `client.dll` memory locations.

<Accordion title="Client Offsets" defaultOpen>
  <Property name="dwLocalPlayerPawn" type="uintptr_t" required>
    Offset to the local player pawn pointer.
  </Property>

  ---

  <Property name="dwViewAngles" type="uintptr_t" required>
    Offset to the view angles pointer.
  </Property>

  ---

  <Property name="dwEntityList" type="uintptr_t" required>
    Offset to the entity list pointer.
  </Property>

  ---

  <Property name="dwViewMatrix" type="uintptr_t" required>
    Offset to the view matrix pointer.
  </Property>
</Accordion>

```lua
local client_base = 0x7FFD12340000 -- Example base address
local local_player_pawn = memory.read_ptr(client_base + offsets.client_dll.dwLocalPlayerPawn)
```

> [!NOTE]
> Additional offsets may be added by Plutonium regularly. Check the cheat console or relevant community forums for a full list of available offsets.
