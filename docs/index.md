# Plutonium CS2 Lua API

Welcome to the Plutonium CS2 Lua API documentation. This documentation provides a comprehensive guide to the functions, tables, and classes available for script developers in Plutonium.

## Getting Started

Plutonium allows you to extend the game's functionality using Lua scripts. You can interact with game memory, draw custom UI elements, and respond to game events.

### API Overview

- **[Globals](/globals)**: Base Lua functions provided by the engine.
- **[Entity](/entity)**: Functions for interacting with game entities (players, pawns, etc.).
- **[Memory](/memory)**: Low-level memory reading and validation.
- **[Draw](/draw)**: Rendering primitives for overlays and UI.
- **[Classes](/classes)**: Object-oriented classes for `Vector3`, `ConVar`, and more.
- **[Offsets](/offsets)**: Pre-defined memory offsets for CS2.
